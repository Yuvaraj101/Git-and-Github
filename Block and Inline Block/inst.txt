Selectors::

element selector : applicable to all tags
class selector: widely used (accessed using dot . and class keyword)
id selector : accessed using # and id keyword
universal selector: * (applies the style property to all the element in the html document)

class name can be same for many elements but id should be only one element per html page

useful for perfect designing

margin -- moves the whole box
padding -- space inside the box
border--needs three property (color,widht,style)

div -division(acts like a container)
